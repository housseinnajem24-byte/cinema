#  Cinéma 

Application web permettant de consulter des films à l’affiche, de rechercher des films
et d’afficher les détails complets d’un film grâce à l’API OMDb.

# Fonctionnalités
- Affichage de films tendances
- Recherche de films en temps réel
- Chargement de résultats supplémentaires
- Page de détails d’un film (résumé, genre, acteurs, note IMDb)

# Technologies utilisées
- HTML
- CSS
- JavaScript
- API OMDb

# Structure du projet
- index.html : page d’accueil
- search.html : page de recherche
- movie.html : page de détails d’un film

# Lancer le projet
Le projet doit être lancé via un serveur HTTP (ex : Live Server sur VS Code).
